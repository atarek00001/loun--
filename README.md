# LOUN Fitting Room 👗

A revolutionary virtual fitting room application where you can see how clothes look on your virtual model Luna.

## Features ✨

- **Virtual Model**: Meet Luna, your digital model with customizable measurements
- **Multiple Poses**: View clothes from 7 different angles
- **Upload Clothing**: Add your own product images to see how they look
- **Clothing Library**: Pre-loaded with 10+ popular clothing items
- **Social Media Formats**: Generate ads in 1:1, 4:5, and 9:16 formats
- **Outfit History**: Save and manage all your generated advertisements
- **Download Ready**: Export high-quality product images instantly

## Getting Started 🚀

### Prerequisites
- Node.js 16+ 
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Open http://localhost:3000
```

### Build for Production

```bash
npm run build
npm start
```

## How to Use 📖

1. **View Luna**: See your virtual model on the main screen
2. **Choose a Pose**: Select from 7 different poses on the right panel
3. **Upload Clothing**: Drag & drop your product image in the upload area
4. **Browse Library**: Select from pre-loaded clothing items
5. **Generate**: Click "Generate Image" to create your advertisement
6. **Download**: Save your generated ad to device

## Tech Stack 🛠️

- **Framework**: Next.js 14
- **UI**: React 18
- **Styling**: Tailwind CSS
- **Deployment**: Vercel Ready
